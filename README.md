# PadLink — Downloads

Official download hub for **PadLink** — turn your Android phone into a low-latency Xbox 360 controller for your Windows PC over Wi-Fi.

➡️ **[Download the latest Windows installer](https://github.com/CheesiePy/padlink-releases/releases/latest)**

The installer (`PadLink-Setup-<version>.exe`) sets up the companion app, the ViGEmBus driver, and a firewall rule for UDP 7777, then shows the pairing QR code.

> **Note:** the exe is unsigned for now — Windows SmartScreen will warn once; choose *More info → Run anyway*.

The PadLink source repository is private for the time being; this repo exists purely to host public releases.
